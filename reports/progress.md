# 進捗ログ（run-over-run）

各スナップショットの見出し指標。決定的評価はゴールデンセット上で毎回再現可能。
LLM合格率など judge 列は `npm run eval:llm -- --report` を走らせた回のみ埋まる。
`dataset=production` は本番50件での実測履歴（README掲載値の出典）、
`dataset=public-sample` は本リポジトリの匿名サンプル10件での計測。
